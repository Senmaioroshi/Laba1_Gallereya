# Лабораторная работа №1 — Gallereya

## Описание

**Gallereya** — Android-приложение для просмотра изображений.  
Приложение позволяет открыть галерею и выбрать картинку.

## Функции

- открытие галереи;
- выбор изображения;
- отображение выбранной картинки на экране.

## Технологии

- Java
- Android SDK
- XML
- ImageView
- Intent

## Разрешение

```xml
<uses-permission android:name="READ_MEDIA_IMAGES" />
```
